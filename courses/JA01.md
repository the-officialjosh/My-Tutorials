  <summary>19. Function Declarations vs. Expressions</summary>
<br>
  There are two ways of writing functions in JavaScript. The first way is called a function declaration and the second way is called a function expression. The main difference between the two is that in a function declaration we can call the function before it is defined in the code. In a function expression we cannot call the function before it is defined in the code.
  
    ```Javascript
    // Function declaration
    function calcAge1(birthYear) { // This is a function
      return 2037 - birthYear; // This returns the value of the expression 2037 - birthYear
    }
  
    const age1 = calcAge1(2001); // This is a function call
    console.log(age1); // This prints the value of the variable age1 in the console
  
    // Function expression
    const calcAge2 = function (birthYear) { // This is a function
      return 2037 - birthYear; // This returns the value of the expression 2037 - birthYear
    }
  
    const age2 = calcAge2(2001); // This is a function call
    console.log(age2); // This prints the value of the variable age2 in the console
    ```
</details>
